# AI Планер дня

Веб-застосунок: вивалюєш усе, що в голові (текстом або голосом) — AI розбирає це на задачі
(пріоритет, час, дедлайн) і формує план на сьогодні.

## Сценарій

1. **Вивалити** — вільний текст або диктування мікрофоном.
2. **Inbox** — AI (Claude Haiku) розбирає дамп на окремі задачі з пріоритетом, оцінкою часу
   та дедлайном. Тут можна поправити будь-яке поле або видалити задачу.
3. **Сьогодні** — за натисканням кнопки формується план на день (з урахуванням бюджету годин,
   пріоритету та дедлайнів), задачі можна відмічати виконаними; невиконане само переноситься
   на завтра.

## Стек

- Next.js 14 (App Router) + TypeScript + Tailwind CSS
- API-роут `/app/api/parse/route.ts` викликає Anthropic API (`claude-haiku-4-5`) для парсингу
- Дані зберігаються локально в браузері (`localStorage`) — без бази даних і авторизації
- Web Speech API для голосового вводу (Chrome/Edge; на iOS Safari може бути недоступно)

## Локальний запуск

```bash
npm install
cp .env.local.example .env.local   # встав свій ANTHROPIC_API_KEY
npm run dev
```

Відкрий http://localhost:3000
