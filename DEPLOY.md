# Деплой: від нуля до живого посилання

Дедлайн — середа, 22.07, 18:00. Розрахунок: усе нижче займає ~20-30 хвилин, якщо йти по
порядку. Роби деплой **першим ділом**, до того як почнеш щось доробляти — порожній сайт у
світі цінніший за ідеальний код на своєму компʼютері.

Тобі знадобляться 3 акаунти: **Anthropic** (API-ключ), **GitHub** (код) і **Vercel** (хостинг).

---

## Крок 1 — Anthropic API ключ ($5 безкоштовно)

1. Відкрий https://console.anthropic.com і зареєструйся (потрібна SMS-верифікація номера).
2. Новий акаунт отримує $5 безкоштовних кредитів — постав на дашборді **Claim**
   (діють 14 днів після кліку, тисни зараз).
3. Settings → API Keys → **Create Key**. Скопіюй ключ (`sk-ant-...`) — він показується
   один раз, встав кудись у нотатки тимчасово.
4. Цього вистачить на весь проєкт і презентацію з великим запасом (один запит парсингу ≈
   500 токенів, $5 = тисячі запитів).

---

## Крок 2 — Заливаємо код на GitHub

Найпростіше — без термінала, прямо в браузері:

1. Зареєструйся на https://github.com, якщо ще немає акаунта.
2. New repository → назви, наприклад, `ai-planner` → **Create repository** (публічний або
   приватний — не важливо, обидва підійдуть для Vercel).
3. На сторінці нового репозиторію натисни **uploading an existing file**.
4. Перетягни в браузер **усі файли й папки** з архіву `ai-planner` (крім `node_modules`,
   якщо він там є — його там не повинно бути).
5. Commit changes.

Якщо звичніше через термінал:

```bash
cd ai-planner
git init
git add .
git commit -m "AI planner MVP"
git branch -M main
git remote add origin https://github.com/ТВІЙ_НІК/ai-planner.git
git push -u origin main
```

---

## Крок 3 — Деплой на Vercel

1. Відкрий https://vercel.com → **Sign up** → увійди через свій GitHub-акаунт (одна кнопка,
   без окремого пароля).
2. **Add New → Project** → обери репозиторій `ai-planner` → **Import**.
3. Framework Preset визначиться сам як **Next.js** — нічого не міняй.
4. Розгорни **Environment Variables** і додай:
   - Name: `ANTHROPIC_API_KEY`
   - Value: твій ключ `sk-ant-...` з кроку 1
5. **Deploy**. Через 1-2 хвилини отримаєш публічне посилання на кшталт
   `https://ai-planner-твійнік.vercel.app`.

⚠️ Якщо пізніше зміниш код на GitHub — Vercel сам передеплоїть сайт при кожному `push`.
Якщо додаєш файли через веб-інтерфейс GitHub (крок 2.3-2.5) — це теж рахується як push.

---

## Крок 4 — Перевір на реальному посиланні (не на локалхості!)

1. Відкрий отримане посилання **з телефона**.
2. Пройди весь сценарій: Вивалити щось голосом → перевір Inbox → сформуй план на сьогодні →
   відміть задачу виконаною.
3. Якщо AI не парсить задачі — перевір, що `ANTHROPIC_API_KEY` збережений у Vercel правильно
   (Project → Settings → Environment Variables) і що ти натиснув Redeploy після його додавання.

---

## Крок 5 — Запиши Loom і здай

1. Запиши відео до 7 хв: яку проблему вирішує продукт → демо (Вивалити → Inbox → Сьогодні)
   → чому таке рішення → (за бажанням) як AI допоміг у розробці.
2. Перевір, що посилання на Loom відкрите для перегляду без логіну.
3. Здай і посилання на Loom, і посилання на задеплоєний Vercel-сайт у форму прийому завдань
   до середи 18:00.

---

## Якщо щось не деплоїться

- **Білий екран / 404** — перевір у Vercel вкладку Deployments → останній деплой → Logs,
  там буде конкретна помилка збірки.
- **AI не відповідає (помилка 500/502 на кнопці «Розібрати»)** — майже завжди означає,
  що `ANTHROPIC_API_KEY` не додано або не збережено у Vercel; додай і натисни Redeploy.
- **Мікрофон не працює** — Web Speech API підтримується в Chrome/Edge на Android і десктопі;
  у Safari на iOS може не працювати — це не критично, текстовий ввід завжди доступний як
  основний шлях.
